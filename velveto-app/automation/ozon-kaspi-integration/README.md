# Ozon & Kaspi Integration Project

Этот проект позволяет автоматически создавать карточки товаров на Ozon, используя данные из МойСклад и изображения из Kaspi.kz.

## Возможности
- Поиск товара в МойСклад по артикулу.
- Извлечение всех фотографий товара из Kaspi.kz.
- Загрузка изображений на ImgBB (надежный хостинг для Ozon).
- Автоматическое определение категорий (Зажигалки, Рюкзаки, Косметички, Зонты, Балаклавы).
- Заполнение обязательных атрибутов Ozon (Бренд, Пол, ТН ВЭД, Размер).

## Установка

1. Установите зависимости:
```bash
pip install requests python-dotenv playwright
playwright install chromium
```

2. Настройте переменные окружения:
Создайте файл `.env` на основе `.env.example` и заполните свои данные.

## Использование

Запустите основной скрипт с указанием артикула:
```bash
python3 main.py <артикул>
```

Пример:
```bash
python3 main.py 109711985
```

## Структура проекта
- `main.py`: Основной скрипт создания карточки.
- `utils/fetch_kaspi_product.py`: Скрапер для Kaspi.kz.
- `utils/check_ozon_task.py`: Проверка статуса импорта на Ozon.
- `utils/get_categories.py`: Поиск категорий Ozon.
- `utils/get_attributes.py`: Получение атрибутов категории.
- `utils/get_dict_values.py`: Поиск значений в словарях Ozon.
