# Ошибки Фронтенда (React/Next.js)

## 1. Hydration Error: "whitespace text nodes cannot be a child of <tr>"

**Симптомы:**
- Экран ошибки в браузере с заголовком "Runtime Error".
- Сообщение указывает на `<table>`, `<thead>`, `<tr>`.
- Причиной называется "extra whitespace" или "whitespace text nodes".

**Причина:**
В HTML внутри тега `<tr>` могут находиться только теги `<th>` или `<td>`.
Если вы оставляете комментарий JSX `{/* ... */}` или пробелы между этими тегами, React может интерпретировать их как текстовые ноды, что недопустимо.

**Пример ошибки:**
```jsx
<tr>
    <th>Col 1</th> {/* Недопустимый комментарий здесь */}
    <th>Col 2</th>
</tr>
```

**Решение:**
Удалите комментарии между ячейками таблицы или перенесите их внутрь тегов `<th>/<td>`.

```jsx
<tr>
    <th>Col 1</th>
    {/* Комментарий можно, если он не генерирует текст, но лучше избегать */}
    <th>Col 2</th>
</tr>
```

**Лучшая практика:** Не писать комментарии прямо внутри структуры `tr` или `table`.
