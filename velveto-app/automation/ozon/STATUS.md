# Статус проекта Ozon Automation

## Сделано
1. **Структура проекта**: Создана папка `ozon-automation` и виртуальное окружение.
2. **Интеграция с МойСклад**:
   - Скрипт `get_ms_products.py` успешно получает товары с остатком > 0 из "Основного склада".
   - Фильтрует товары без изображений.
   - Скачивает изображения локально.
3. **Поиск аналогов**:
   - Скрипт `find_ozon_match.py` использует Playwright для поиска товаров в Яндекс.Картинках по фото.
   - Успешно находит ссылки на Ozon для некоторых товаров.
4. **Парсинг Ozon (WIP)**:
   - Скрипт `scrape_ozon_details.py` написан, но сталкивается с защитой Ozon (Access Denied). Требуется доработка (прокси, куки).
