# Инструкция по переносу "Мозгов" (Worker) в Облако

Чтобы система работала 24/7 без вашего включенного компьютера, нужно перенести скрипт-обработчик (Worker) на удаленный сервер (VPS). Самый простой вариант — сервер за $5/мес (например, DigitalOcean, Hetzner, Aeza).

Я подготовил всё для "упаковки" вашего кода.

### Шаг 1. Аренда Сервера
1.  Арендуйте любой VPS сервер с **Ubuntu 22.04**.
2.  Вам пришлют IP адрес и пароль (root).

### Шаг 2. Копирование файлов
Вам нужно скопировать папку проекта на сервер. Это можно сделать через FileZilla или команду `scp`.
Нужны файлы:
*   `Dockerfile`
*   `docker-compose.yml`
*   `requirements.txt`
*   Все папки `moysklad-web` и скрипты `.py`.

### Шаг 3. Запуск на сервере
Подключитесь к серверу через терминал (SSH) и выполните команды:

1.  **Установите Docker:**
    ```bash
    curl -fsSL https://get.docker.com | sh
    ```

2.  **Запустите Worker:**
    Находясь в папке с файлом `docker-compose.yml`:
    ```bash
    docker compose up -d --build
    ```

### Итог
*   Сайт (Vercel) будет создавать задачи.
*   Ваш сервер (Docker) будет их видеть, запускать браузер внутри себя и обновлять базу.
*   Вы можете выключить свой ноутбук.
